Este é um parágrafo em destaque com fundo amarelo.

Este é um parágrafo normal.

Neste exemplo, estamos usando a propriedade background-color para dar ao parágrafo com a classe .destaque um fundo amarelo. A propriedade padding também é usada para adicionar um espaçamento interno ao parágrafo, para que o fundo amarelo não fique muito próximo ao texto. Isso resultará em um parágrafo com fundo amarelo e um espaçamento interno para melhor legibilidade.
